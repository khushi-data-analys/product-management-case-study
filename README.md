# 📊 Product Management Case Study

## 📌 Overview
This project focuses on improving user engagement in a mobile application.

## 🚀 Problem
User engagement is decreasing due to poor onboarding and feature discovery.

## 💡 Solution
- Improved onboarding
- Push notifications
- Feature optimization

## 📊 Metrics
- DAU
- Retention Rate
- Session Duration

## 📈 Outcome
Better engagement and user retention
